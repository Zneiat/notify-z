A Simple Notify jQuery Plug-ins.
一个简约美观的 jQuery 前端通知插件

# 使用方法

### 引入
```html
<link href="/assets/notify.css" rel="stylesheet">
<script src="/assets/notify.js"></script>
```
P.S. 其实代码不多，可以放入自己的项目里.

### 调用
```javascript
$.notify.success('成功\n上传成功');
$.notify.error('错误\n上传失败');
$.notify.info('信息');
$.notify.warning('警告');
```
